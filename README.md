# 4目並べ (Connect-4)

## バージョン
`clang version 11.0.3`<br>
`GNU Make 3.81`

## 実行方法
```
$ git clone https://github.com/Toshiki-Nakamura/dena-review.git
$ cd dena-review
$ make
$ ./connect_four
```

## 操作/ルール

プレイヤーはどの列(1~7)にコマを置くか選択します
縦か横か斜めに，先に４つ並べた方の勝ち
